'table' tags
  • 'thead' = table row
  • 'tr' = table row
  • 'th' = Heading Name for that row
  • 'th' (with colspan attribute) for a table name that spans all specified number of colums :: th colspan="3"
  • 'tbody' = table body (contains actual tabular data)
  • 'tr' = table row
  • 'td' = table data for that row
  • 'thead' = table head (contains the table HEADINGS)
  • 'tr' = table row

'table' attributes:
  • table border="Xpx"
